# Earthquake Analysis Dashboard using Python

## Overview
This project analyzes and visualizes recent global earthquake activity using real-time data from the United States Geological Survey (USGS). It combines statistical analysis with interactive geospatial visualization.

## Features
- Fetches live earthquake data from USGS
- Interactive world map with color-coded earthquake markers
- Basic statistical analysis
- Magnitude distribution histogram
- Top 10 strongest earthquakes

## Technologies Used
- Python
- Google Colab
- Pandas
- Folium
- Matplotlib

## Data Source
United States Geological Survey (USGS)
https://earthquake.usgs.gov/

## Repository Contents
- `Earthquake_Analysis.ipynb` – Project notebook
- `Project_Report.pdf` – Project report
- `earthquake_dashboard.html` – Interactive map (optional)
